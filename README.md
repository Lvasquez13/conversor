# conversor
Este proyecto tiene como objetivo demostrar de forma práctica cómo desarrollar una aplicación web interactiva utilizando HTML, CSS y JavaScript. Se trata de un conversor de divisas que permite al usuario ingresar un valor en una moneda (por ejemplo, dólares o pesos) y obtener su equivalente en otra, mostrando el resultado de manera dinámica.
El programa busca reforzar conceptos fundamentales del desarrollo web, como:

La vinculación de un archivo JavaScript externo con un documento HTML.

El uso de funciones para realizar cálculos y modificar el DOM.

La aplicación de buenas prácticas de modularización, separando la lógica del diseño y cumpliendo con las normas básicas de legibilidad del código.

La interactividad con el usuario mediante entradas (input), botones y actualizaciones visuales en tiempo real.

Además, el proyecto tiene un propósito educativo, orientado a que los estudiantes comprendan cómo se integran las distintas capas de una página web (estructura, estilo y comportamiento) para crear una aplicación funcional y comprensible.
